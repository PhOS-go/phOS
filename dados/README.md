# dados/

Coloque aqui os arquivos que você quer que o sistema analise: CSV, XLSX, PDF,
planilhas, exports de métricas.

Quando rodar `/analisar-dados` ou `/relatorio`, o sistema lê o que estiver nesta
pasta (ou no arquivo que você indicar) e devolve o resultado em `entregas/relatorios/`.

> Arquivos com dados sensíveis: confira o `.gitignore` antes de `/salvar` se não
> quiser versioná-los.
