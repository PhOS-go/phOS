# scripts/

Utilitários do dia a dia, pequenos scripts que automatizam tarefas repetidas
(gerar imagem, renderizar HTML em PNG, postar, converter arquivo).

Quando uma skill precisar de um script de apoio, ele é criado aqui. Você também
pode pedir ao sistema pra criar um script novo a qualquer momento.

Mantenha um script por função, com nome claro do que ele faz.
